## 💡 学习建议

1. **先用H2内存数据库**：快速上手，无需安装MySQL
2. **理解ORM概念**：对象关系映射是核心思想
3. **多写SQL对比**：看JPA生成的SQL，理解底层原理
4. **先学JPA再学MyBatis**：JPA更简单，MyBatis更灵活
5. **实战为主**：每个知识点都在书签项目中应用

## 🛠️ 环境准备

开始学习前，确保已安装：
- MySQL 8.0+ 或 PostgreSQL 14+（推荐先用H2）
- MySQL Workbench 或 DBeaver（数据库客户端）
- 已完成第一阶段学习

验证安装：
```bash
mysql --version   # 如果安装了MySQL
```

## 🗄️ 数据库表设计预览

```sql
-- 用户表
users (id, username, email, password, created_at)

-- 书签表
bookmarks (id, title, url, description, user_id, created_at)

-- 标签表
tags (id, name, color)

-- 书签-标签关联表（多对多）
bookmark_tags (bookmark_id, tag_id)

-- 收藏夹表
folders (id, name, user_id, parent_id)

-- 书签-收藏夹关联表
bookmark_folders (bookmark_id, folder_id)
```
