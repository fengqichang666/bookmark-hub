## 💡 学习建议

1. **边学边敲代码**：不要只看文档，每个示例都要自己敲一遍
2. **对比前端知识**：文档中会用前端概念类比，帮助快速理解
3. **遇到问题先查文档**：每个文档都有"常见问题"部分
4. **完成实战项目**：最后4天的实战项目是检验学习成果的关键
5. **使用IntelliJ IDEA**：强烈建议用IDEA而不是VSCode

## 🛠️ 环境准备

开始学习前，确保已安装：
- JDK 17+ (推荐用 Amazon Corretto 或 Oracle JDK)
- Maven 3.8+
- IntelliJ IDEA Community/Ultimate
- Postman (测试API用)

验证安装：
```bash
java -version   # 应该显示 17 或更高版本
mvn -version    # 应该显示 3.8 或更高版本
```

## 📌 项目介绍：书签管理系统

我们将构建一个简单但完整的书签管理系统，功能包括：
- 添加书签（标题、URL、分类）
- 查看所有书签
- 按分类筛选书签
- 更新书签信息
- 删除书签

这个项目会贯穿整个学习过程，每学一个知识点都会在项目中应用。
