## 📖 如何使用这些文档

### 1. 按顺序学习
从第一阶段的 `00-学习路线图.md` 开始，按照文档编号顺序学习。

### 2. 边学边做
每个文档都包含：
- 📝 理论讲解（对比前端知识）
- 💻 代码示例
- 🎯 实战练习
- ❓ 常见问题

### 3. 完成实战项目
两个阶段都有完整的书签管理系统实战：
- 第一阶段：内存版本（List存储）
- 第二阶段：数据库版本（JPA持久化）

### 4. 检查学习成果
每个阶段都有检查清单，确保掌握核心知识点。

## 💡 学习建议

1. **每天2-3小时**：保持学习节奏，不要贪多
2. **动手实践**：每个示例都要自己敲一遍
3. **理解原理**：不要死记硬背，理解为什么这样做
4. **对比前端**：利用已有的前端知识快速理解
5. **遇到问题先查文档**：每个文档都有"常见问题"部分
